# Instrucciones Para El Equipo

La idea es que el equipo edite un Google Sheet, no codigo.

Como debe verse el archivo:

- Una pestana por tema
- No borrar los encabezados
- Una fila = una cosa
- Escribir `SI` o `NO` cuando la columna lo pida
- Si algo no aplica, dejar vacio
- No escribir en ingles
- No usar abreviaturas raras si no son necesarias
- Usar las listas desplegables cuando aparezcan
- Mirar siempre las columnas `Interpretacion IA`, `Estado IA` y `Observaciones IA`

Pestanas sugeridas:

- `sedes`
- `profesionales`
- `examenes`
- `reglas_de_cobertura`
- `preguntas frecuentes`

Texto simple para poner en una pestana `LEEME` del Google Sheet:

```text
Hola equipo.

Este archivo alimenta al bot.
Por favor escriban solo informacion vigente y confirmada.

Reglas simples:
1. No borrar encabezados.
2. Una fila por sede, doctor, procedimiento o regla.
3. Escribir SI o NO cuando se pida.
4. Si un dato no aplica, dejar vacio.
5. No escribir codigos ni terminos tecnicos si no hacen falta.
6. Si una informacion no esta confirmada, no ponerla.
7. No prometer horarios ni precios si no estan validados.
8. Si la columna Estado IA dice Revisar, corregir la fila antes de seguir.
```
